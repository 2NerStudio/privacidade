Política de Privacidade – Seu Bot

Última atualização: 10 de maio de 2025

1. Informações que coletamos

Coletamos os seguintes dados para o funcionamento do bot:

  • ID de usuário do Discord
  • ID de servidor e canal
  • Mensagens com comandos
  • Configurações de servidor e preferências

2. Como usamos os dados

Utilizamos os dados para:

  • Executar comandos do bot
  • Salvar configurações por servidor
  • Melhorar a experiência do usuário

3. Armazenamento e segurança

Os dados são armazenados com segurança em nossos servidores. Apenas os administradores têm acesso a eles.

4. Compartilhamento de dados

Não compartilhamos ou vendemos nenhum dado coletado. Dados só serão divulgados em caso de obrigação legal.

5. Direitos dos usuários

Você pode solicitar a exclusão ou visualização de seus dados entrando em contato conosco.

6. Alterações

Esta política pode ser atualizada. Notificaremos as mudanças aqui e no servidor oficial do bot.
